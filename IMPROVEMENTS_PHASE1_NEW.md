## 🎯 残りの改善提案（優先度順）

### P0: 収益化

#### 1. Amazon書籍の動的表示・配置最適化
**目的**: 計算結果に応じて関連書籍を自動推奨→クリック率向上

**仕様**:
- 現在: Tips タブにのみ Amazon 書籍リンク配置
- 改善: 計算結果の「関連情報・詳細ガイド」セクション内に、計算内容に応じた Amazon 書籍を動的提示
  - 高所得者向け（500万円以上）→「相続税対策本」「投資本」
  - 青色申告希望者→「青色申告の本」「簿記入門」
  - iDeCo活用者→「投資・資産形成本」
  - ふるさと納税活用者→「ふるさと納税・お得本」
- 最大3冊を「あなたにおすすめの書籍」として表示

**実装タスク**:
- HTML 結果セクションに条件分岐で Amazon アフィリエイトリンク（書籍）を追加
- JavaScript で条件判定（年収帯、選択項目、金額）に応じてリンク動的生成
- GA/GTM で「書籍」クリック追跡設定

**DoD**:
- [ ] 計算結果表示時に、条件に応じた 1〜3冊の書籍がおすすめされる
- [ ] モバイル・PC で見やすくレイアウトされている
- [ ] GTM で「book_click」が追跡される

### P1: SEO・集客

#### 2. 副業タイプ別・内部リンク強化
**目的**: 副業タイプごとに特化したコンテンツ・ツールへの内部リンク→集客・ドメイン評価向上

**仕様**:
- 現在: 「関連情報」には個別リンク（ブログ記事・FIRE計算機）のみ
- 改善: 副業タイプに応じて異なるリンクを提示
  - ブログ・アフィリエイト→「ブログ収益化ガイド」「SEO診断ツール」へのリンク
  - せどり・物販→「商品利益計算ツール」「Amazon 出品ガイド」へのリンク
  - ハンドメイド→「販売手数料計算ツール」「著作権ガイド」へのリンク
  - 家庭教師・講師→「時給相場リサーチツール」「集客ガイド」へのリンク
  - Stock 投資→「FIRE 計算機」「資産形成ガイド」へのリンク
- 各リンクはカード形式で 2〜3 個を表示（関連情報セクションを拡充）

**実装タスク**:
- JavaScript で副業タイプ判定時に、対応するリンクセット（配列）を選択
- HTML 結果エリア「関連情報・詳細ガイド」セクションに、タイプ別リンク 3〜5 個を動的生成

**DoD**:
- [ ] 副業タイプごとに異なるリンクが表示される
- [ ] 全 11 タイプで 3 個以上のリンクが表示される
- [ ] GTM で「related_link_click」が追跡される

### P2: ユーザー継続

#### 3. 確定申告チェックリスト機能
**目的**: 「次のステップ」を具体的なタスク化→行動喚起・再訪

**仕様**:
- 計算結果に基づいて、「📋 あなたの確定申告・節税チェックリスト」を表示
- ロジック:
  - **確定申告が必要な場合**: 
    - [ ] 帳簿記帳・領収書整理（期限: 1月31日まで）
    - [ ] 青色申告申請書提出（初年度のみ）
    - [ ] e-Tax 用マイナンバーカード準備
    - [ ] 確定申告書作成（freee/MF 推奨）
    - [ ] 税務署窓口 or e-Tax で提出（2月15日〜3月15日）
  - **確定申告不要だが住民税申告が必要**: 上記から「確定申告」タスクを除く
  - **iDeCo や ふるさと納税 を利用**: 対応タスク追加
- チェックボックス（localStorage で保存可能）、完了時に進捗バー表示

**実装タスク**:
- calcResult 内に HTML チェックリストを動的生成
- localStorage で「checked_tasks_sidejob」に保存・復元
- CSS で進捗バー（完了率）を視覚化

**DoD**:
- [ ] 確定申告必要時に 4〜6 個のタスクが表示される
- [ ] チェックボックスがクリック可能で localStorage に保存される
- [ ] モバイル・PC で見やすくレイアウトされている

---

#### 4. 計算履歴・シナリオ比較機能（オプション）
**目的**: 複数シナリオの保存→「もし月10万円だったら？」の比較検討

**仕様**:
- 計算結果下に「💾 このシナリオを保存」ボタン
  - クリックで、現在の入力値＋計算結果を localStorage に保存
  - 保存時に「2024年6月・フリーランス・月20万」など自動命名
- 新規計算時に「📊 保存済みシナリオ」タブを追加
  - 保存結果を最大 5 件表示（表：副業タイプ、月収入、手取り、税率）
  - 各結果をクリックで当時の設定を復元
  - 削除機能あり（localStorage から削除）

**実装タスク**:
- 「保存」ボタンのイベントリスナー実装（sjSaveScenario 関数）
- localStorage に JSON 形式で保存（maxItems: 5、最古のものから削除）
- 「保存済みシナリオ」タブのパネル追加・表示ロジック

**DoD**:
- [ ] 計算結果を保存・復元できる
- [ ] 最大 5 件まで保存可能
- [ ] localStorage 容量内に収まる（<1MB）
- [ ] ページリロード後も保存が維持される
